## Getting Started

- Clone the repo
- Create file `.env.local` and insert: `NEXT_API_URL='https://sujeitoprogramador.com'`
- Open terminal, run:
`
npm run dev
`

- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
